A **modern real estate listing and classifieds platform** with **advanced filtering** and a fully **responsive user interface**. Built with **Next.js 13 (App Router)**, **TypeScript**, and **MongoDB**, this project allows managing and displaying listings with multiple dynamic filters such as **category, transaction type, price range, and location**.

---

## ‚ú® Features

* Post and display **Villa, Apartment, Shop, and Office** listings
* **Live filtering** of listings by:

  * Category (`category`)
  * Transaction type (`buy`, `rent`, `mortgage`)
  * City and location (`city`, `location`)
  * Price range (`minPrice`, `maxPrice`)
* **Responsive and modern UI** using **Tailwind CSS** and **Radix UI**
* **Server + Client Components** for SSR and seamless client-side filtering
* API routes built with **Next.js Route Handlers** connected to **MongoDB**
* Listing images with **fallback support**
* Sorted listings by **creation date**
